# 📊 성은이와 병철이의 가계부 대시보드 (Household Ledger Dashboard)

이 프로젝트는 우리 가족의 가계부 내역을 직관적이고 아름답게 시각화하여 자산 흐름을 한눈에 관리할 수 있게 도와주는 **대화형 웹 대시보드**입니다.

---

## 🔗 배포 링크
- **GitHub Pages**: [https://jbc6265.github.io/household-ledger/](https://jbc6265.github.io/household-ledger/)

---

## ✨ 주요 기능
* **프리미엄 UI/UX**: 깊이감 있는 글래스모피즘 디자인과 우주 톤 다크 테마를 차용하여 모던하고 감각적인 화면을 제공합니다.
* **인터랙티브 차트 (Chart.js)**: 
  * 월별 수입 및 지출 추이 비교
  * 고정비 대비 변동비의 월별 추이 및 점유율 분석
  * 주유비 및 하이패스 통행료 현황 분석
  * 결제 카드사별 점유 비율 분석
  * 대분류별 지출 패턴 시각화
* **드래그 앤 드롭 업로드**: 웹 브라우저 화면에 최신 가계부 CSV나 엑셀 파일을 떨어뜨려 로컬 데이터를 즉시 갱신할 수 있습니다.
* **로컬 보안**: 가벼운 패스워드 검증 화면을 제공하여 개인 정보가 외부로 노출되지 않도록 도와줍니다.

## 📂 파일 구조 및 역할

| 파일명 | 역할 설명 |
| :--- | :--- |
| `generate_dashboard.py` | 엑셀(.xlsx) 데이터를 읽고 정형화된 JSON으로 변환하여 `template.html`에 주입해 `index.html`을 자동 빌드하는 Python 스크립트 |
| `template.html` | 웹 대시보드의 원본 마스터 템플릿 (코드 빌드용 뼈대) |
| `index.html` | 가계부 데이터가 주입 완료되어 실제로 서비스되는 정적 대시보드 웹페이지 |
| `main.py` | 로컬 및 클라우드 배포 시 완성된 `index.html`을 실행해주는 가벼운 Flask 웹 서버 |
| `vercel.json` | Vercel 클라우드 서버리스 배포를 위한 설정 파일 |
| `Procfile` | Heroku 배포 시 WSGI 웹 서버(`gunicorn`) 구동 규칙 선언 파일 |
| `requirements.txt` | 프로젝트 구동에 필요한 Python 의존성 패키지 명세 |

---

## 🛠️ 로컬 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 가계부 데이터 빌드 (index.html 생성)
`generate_dashboard.py`를 실행하여 지정된 경로의 최신 엑셀 내역을 파싱하고 `index.html`을 컴파일합니다.
```bash
python generate_dashboard.py
```

### 3. Flask 로컬 웹 서버 가동
```bash
python main.py
```
* 서버가 가동되면 웹 브라우저에서 `http://localhost:8080`으로 접속하여 대시보드를 사용할 수 있습니다.

## 🧠 가계부 분류 및 분석 규칙

### 1) 자동 필터링 제외 대상
* **춘식이페어 후불교통비**: 중복 계산 방지를 위해 전체 통계에서 제외합니다.
* **테슬라 계약금/취소 거래**: 일시적인 대형 입출금 건(상쇄되는 거래)은 생활비 흐름 분석을 왜곡하므로 통계에서 제외합니다.

### 2) 고정비 분류 기준
* **카테고리**: `주거/통신` 분류 내역 전체.
* **키워드**: 거래 내용에 `와우멤버십`, `구독`, `흥화`, `교보`, `코웨이렌탈`, `Apple Inc`가 포함된 경우 자동 고정비 분류.
* *단, `(주)동남에너지` 결제 건은 주거/통신 카테고리에 속해 있어도 변동비(주유비)로 우회 처리됩니다.*

### 3) 차량 관련 비용 분류
* **주유비**: `자동차` 분류이면서 내용에 `주유소`, `에너지`, `오일뱅크`, `석유`, `주유` 등이 포함된 거래.
* **통행료**: 주유비가 아니면서 결제수단이 `현대카드ZERO Edition2 포인트형 하이패스`이거나 내용에 `대교`, `고속도로`, `도로` 등이 포함된 거래.
